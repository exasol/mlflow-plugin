=================================

<https://mlflow.org/docs/latest/self-hosting/architecture/artifact-store/#setting-a-default-artifact-location-for-logging>`_
for specifying the BucketFS Artifact Store. You can define this location
either when starting the MLflow server or when creating an MLflow
*experiment*.
