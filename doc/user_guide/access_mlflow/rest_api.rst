REST API UDFs
=============

Exasol MLflow Plugin contains a list of `User Defined Scripts
<https://docs.exasol.com/db/latest/database_concepts/udf_scripts.htm>`_ (UDFs)
to access MLflow's REST API and get the results as SQL tables in Exasol.

The UDFs require the plugin's SLC to be deployed and activated, see
:ref:`install_slc`.
